# Sugestão de Design System - AjudaPet

Aqui está o conceito visual que geramos unindo o **Verde Esmeralda Escuro** (para confiança e saúde) com o **Laranja** (para alertas, energia e chamadas para ação).

![AjudaPet Dashboard Concept](C:/Users/juhco/.gemini/antigravity/brain/be96d1f1-135a-4d7f-a15a-0af68b4e4436/ajudapet_dashboard_mockup_1780796910181.png)

## Pontos Chave do Design:
*   **Fundo Claro:** Facilita a leitura e traz uma estética "clean" de clínica.
*   **Verde Esmeralda:** Traz seriedade para os menus e títulos.
*   **Laranja Vibrante:** Usado para chamar atenção (ex: botões de "Pesar Pet", "Vacinas Atrasadas").
*   **Formas Arredondadas:** Traz uma sensação mais amigável, já que estamos lidando com pets.

Me diga o que acha dessa direção visual para começarmos a construir o CSS (Tailwind)!
